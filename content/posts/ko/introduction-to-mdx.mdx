MDX는 Markdown 파일에서 JSX를 작성할 수 있게 해주는 강력한 도구입니다. 이
글에서는 MDX의 기본 사항을 소개합니다.

## MDX란 무엇인가요?

MDX는 Markdown 파일에서 JSX를 작성할 수 있게 해주는 파일 형식입니다. 이는 React
컴포넌트를 사용하고, 다른 컴포넌트를 가져오며, 심지어 Markdown 파일 내에서
인라인 JSX를 작성할 수 있다는 것을 의미합니다.

```
export async function getSource() {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  return data;
}
```

## MDX를 사용하는 이유

MDX는 대화형 문서, 블로그 게시물 및 일반 텍스트 이상이 필요한 기타 콘텐츠를
작성하는 데 탁월한 방법입니다. MDX를 사용하면 대화형 컴포넌트, 코드 스니펫 및
기타 동적 콘텐츠를 Markdown 파일에 쉽게 포함할 수 있습니다.

<Counter />

## MDX 사용 방법

MDX를 사용하려면 정적 사이트 생성기나 빌드 도구용 플러그인을 설치해야 합니다.
예를 들어, Gatsby를 사용하는 경우 `gatsby-plugin-mdx` 플러그인을 사용하여
프로젝트에서 MDX 지원을 활성화할 수 있습니다.

플러그인을 설치한 후에는 `.mdx` 확장자로 파일을 만들고 그 안에 JSX를 작성하여
MDX 파일을 작성할 수 있습니다.
