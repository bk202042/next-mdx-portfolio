복잡한 해외 부동산 정보를 단순하고 직관적으로 제공하는 특화 플랫폼을 개발했습니다. 베트남 주요 도시(호치민, 하노이, 다낭, 하이퐁 등)의 부동산 정보를 한국어로 통합 제공하여 언어 장벽과 정보 접근성 문제를 해결했습니다.

- **한국인 거주자 특화 디자인 설계**: 한국식 사용자 경험과 선호도를 반영한 직관적 인터페이스
- **복잡한 베트남 부동산 시장 정보의 체계적 단순화**: 현지 복잡한 부동산 규제와 거래 정보를 알기 쉽게 정리
- **실시간 매물 정보 자동화 시스템 도입**: 최신 매물 정보 자동 업데이트 및 알림 서비스 구현


## 기술 스택

이 프로젝트는 다음과 같은 최신 기술을 활용하여 개발되었습니다:

- **Next.js**: 서버 사이드 렌더링 및 정적 사이트 생성을 위한 React 프레임워크
- **React**: 사용자 인터페이스 구축을 위한 JavaScript 라이브러리
- **Tailwind CSS**: 맞춤형 디자인을 위한 유틸리티 우선 CSS 프레임워크
- **MongoDB**: 유연한 스키마로 부동산 데이터를 저장하는 NoSQL 데이터베이스
- **Vercel**: 웹 애플리케이션 배포 및 호스팅 플랫폼

## 주요 기능

- **직관적인 검색 필터**: 지역, 가격, 면적, 방 개수 등 다양한 조건으로 매물 검색
- **상세한 매물 정보**: 사진, 도면, 위치, 시설 등 상세 정보 제공
- **실시간 알림 서비스**: 새로운 매물 등록 시 사용자 조건에 맞는 알림 발송
- **다국어 지원**: 한국어와 영어 인터페이스 제공
- **모바일 최적화**: 반응형 디자인으로 모든 기기에서 원활한 사용 경험
