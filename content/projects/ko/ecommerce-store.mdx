이 프로젝트는 NextJs와 Stripe로 구축된 풀스택 이커머스 스토어입니다. 결제 처리를
위해 `@stripe/stripe-js` 패키지를 사용하고, 번들 크기 분석을 위해
`@next/bundle-analyzer` 패키지를 사용합니다.

## 주요 기능

- **Stripe 결제**: Stripe API를 사용한 결제 처리
- **상품 카탈로그**: 그리드 레이아웃으로 상품 표시
- **장바구니**: 상품을 장바구니에 추가하고 결제 진행
- **사용자 인증**: 회원가입 및 로그인 기능
- **주문 내역**: 주문 내역 조회 및 배송 추적

## 사용 기술

- **NextJs**: 정적 및 동적 웹사이트 구축을 위한 React 프레임워크
- **Stripe**: 온라인 비즈니스를 위한 결제 처리 플랫폼
- **Tailwind CSS**: 맞춤형 디자인을 위한 유틸리티 우선 CSS 프레임워크
- **Vercel**: 정적 사이트 및 서버리스 함수를 위한 클라우드 플랫폼

## 시작하기

이 프로젝트를 시작하려면 저장소를 클론하고 의존성을 설치하세요:

```bash
git clone
cd ecommerce-store
npm install
```

의존성 설치가 완료되면 개발 서버를 실행할 수 있습니다:

```bash
npm run dev
```

개발 서버는 `http://localhost:3000`에서 시작됩니다.

## 배포

이 프로젝트를 배포하려면 Vercel을 사용할 수 있습니다. GitHub 저장소를 Vercel에
연결하고 클릭 한 번으로 프로젝트를 배포할 수 있습니다.

## 결론

이 이커머스 스토어는 NextJs와 Stripe로 풀스택 애플리케이션을 구축하는 방법을
보여주는 훌륭한 예시입니다. Stripe API를 NextJs와 통합함으로써 고객에게 원활한
쇼핑 경험을 제공하고 결제를 안전하고 효율적으로 관리할 수 있습니다.
