NextJs와 MDX를 사용하여 구축된 포트폴리오 웹사이트입니다. `@next/mdx` 패키지를
사용하여 NextJs 페이지에서 MDX 콘텐츠를 렌더링합니다.

## 주요 기능

- **MDX 지원**: MDX 형식으로 콘텐츠 작성.
- **Syntax Highlighting**: 코드 블록에 Syntax Highlighting 적용.
- **커스텀 컴포넌트**: MDX 파일에서 커스텀 컴포넌트 사용.
- **반응형 디자인**: 모든 디바이스에서 반응형으로 작동.

## 사용 기술

- **NextJs**: 정적 및 동적 웹사이트를 구축하기 위한 React 프레임워크.
- **MDX**: Markdown 파일에 JSX를 작성할 수 있는 파일 형식.
- **Tailwind CSS**: 커스텀 디자인을 구축하기 위한 유틸리티-퍼스트 CSS
  프레임워크.
- **Vercel**: 정적 사이트와 서버리스 함수를 위한 클라우드 플랫폼.

## 시작하기

이 프로젝트를 시작하려면, 리포지토리를 클론하고 의존성을 설치하세요:

```bash
git clone [invalid URL removed]
cd next-mdx-portfolio
npm install
```

의존성이 설치되면, 개발 서버를 실행하세요:

```bash
npm run dev
```

개발 서버는 `http://localhost:3000`에서 시작됩니다.

## 배포하기

이 프로젝트를 배포하려면, Vercel을 사용할 수 있습니다. GitHub 리포지토리를
Vercel에 연결하고 한 번의 클릭으로 프로젝트를 배포할 수 있습니다.

## 결론

이 포트폴리오 웹사이트는 NextJs와 MDX를 사용하여 동적 콘텐츠를 포함한 현대적인
웹사이트를 구축하는 방법의 좋은 예입니다. NextJs와 MDX의 강력한 기능을 결합하여,
유지 보수와 업데이트가 쉬운 상호 작용적이고 매력적인 웹사이트를 만들 수
있습니다.
