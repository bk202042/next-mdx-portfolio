This is a full-stack ecommerce store built with NextJs and Stripe. It uses the
`@stripe/stripe-js` package to handle payments and the `@next/bundle-analyzer`
package to analyze the bundle size.

## Features

<div className="my-8 rounded-lg bg-gradient-to-r from-blue-50 to-indigo-50 p-6 shadow-sm border border-blue-100">
  <div className="flex items-start">
    <div>
      <h3 className="text-lg font-medium text-blue-800">Key Capabilities</h3>
      <div className="mt-2 text-blue-700">
        <p className="mb-3">This platform offers a complete end-to-end ecommerce solution with the following features:</p>
      </div>
    </div>
  </div>
</div>

- **Stripe Payments**: Accept payments using the Stripe API.
- **Product Catalog**: Display products in a grid layout.
- **Shopping Cart**: Add products to the shopping cart and checkout.
- **User Authentication**: Allow users to sign up and log in.
- **Order History**: View order history and track shipments.

## Technologies

This project is built with a modern tech stack optimized for performance and developer experience:

- **NextJs**: A React framework for building static and dynamic websites.
- **Stripe**: A payment processing platform for online businesses.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Vercel**: A cloud platform for static sites and serverless functions.

## Getting Started

To get started with this project, you can clone the repository and install the dependencies:

<pre className="my-6 rounded-lg bg-gray-900 p-4 overflow-x-auto">
  <code className="text-sm text-gray-100">git clone
cd ecommerce-store
npm install</code>
</pre>

Once the dependencies are installed, you can run the development server:

<pre className="my-6 rounded-lg bg-gray-900 p-4 overflow-x-auto">
  <code className="text-sm text-gray-100">npm run dev</code>
</pre>

The development server will start at `http://localhost:3000`.
